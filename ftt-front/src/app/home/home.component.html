Projeto FTT

Requisitos

Você deverá criar uma API Java utilizando o framework Spring Boot e uma interface gráfica utilizando TypeScript com o framework Angular 2+. Esta aplicação deve ser capaz de persistir dados em banco SQL.

História de usuário

Sendo dono de um site de anúncios de carros quero poder manter uma lista de garagens, para poder cadastrar carros nas garagens.

Sendo dono de um site de anúncios de carros quero poder manter uma lista de carros que tenho a venda para que meus clientes possam visualizar a lista dos mesmos.

Critérios

1. Utilização de boas práticas de programação(Ex. Organização do código, nomenclatura, separação do código, identação, etc).

2. Mapeamento das rotas de navegação REST da API.

3. Cadastro, exclusão, alteração, visualização(através do campo modelo) e listagem dos carros.

4. A garagem terá as seguintes características:

  • Nome - texto
  • Endereço(logradouro, complemento, bairro, cidade, estado, pais)

    • 5. O carro terá as seguintes características:

  • Fabricante - texto
  • Modelo - texto
  • Versão - texto
  • Descrição - texto
  • Cor - texto
  • Valor - texto
  • Garagem - select

    • 6. Todos os campos serão obrigatórios.

    7. A conclusão total do projeto não é obrigatória pois cada etapa possui uma pontuação separada.
